# react-payments

### 카드 번호 입력 및 식별

- [ ]  사용자가 입력하는 카드 번호를 실시간으로 파악
- [ ]  Visa나 MasterCard에 해당하면 해당 브랜드의 로고를 UI에 표시한다.
- [ ]  숫자에 대한 유효성 검사 진행
    - [ ]  입력은 숫자만 가능
    - [ ]  유효하지 않은 경우 피드백 제공

### 카드 유효기간 입력

- [ ]  연월을 입력한다.
- [ ]  연월에 대한 유효성 검사
    - [ ]  범위 내에서만 입력 가능
    - [ ]  숫자만 입력 가능

### 실시간 프리뷰 업데이트

- [ ]  한 input의 값이 채워지고 유효성 검사가 통과 되면 카드 프리뷰에 노출시킨다.
    - [ ]  고유 식별 번호에 따라서 카드 로고가 변경된다.
        - [ ]  Visa: 4로 시작하는 16자리 숫자
        - [ ]  MasterCard: 51~55로 시작하는 16자리 숫자
