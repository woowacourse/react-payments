## 1단계 구현 기능 목록

- 재사용가능한 컴포넌트 추가
  - card
  - button
  - inputBox
- 입력 폼 추가
- 입력 값 검증
  - password에는 number만 들어올 수 있다
  - 들어올 수 있는 자릿수는 정해져있다
  - 만료되는 달은 1~12 사이의 숫자이다
  - 만료되는 년도는 22년 이후의 년도이다
- 포커스
  - 정해진 입력 길이 충족 시 다음 입력으로 넘어간다
- 다음 버튼
  - 모든 폼을 작성 시 다음 버튼이 활성화된다
